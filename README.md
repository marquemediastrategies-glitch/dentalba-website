# Albania Dental Tourism Organic Lead System

This workspace contains a free-lead generation workflow for dental tourism in Albania, focused on European patients looking for high-quality dental treatment at more affordable prices.

Important: no SEO workflow can guarantee success. Search rankings and leads depend on competition, site authority, reviews, trust assets, execution quality, and time. This system is designed to maximize the odds by targeting high-intent keywords, publishing consistently, capturing leads properly, and following up fast.

## Start Here

1. Replace the placeholder WhatsApp number in `assets/js/site.js`.
2. Add the final domain to HTML canonicals, hreflang tags, `robots.txt`, and `sitemap.xml` after the domain is chosen.
3. Replace generated placeholder imagery with real DentAlba clinic/team/case photos when available.
4. Submit the site to Google Search Console and Bing Webmaster Tools after launch.
5. Track every inquiry in `tracking/lead-tracker.csv`.
6. Review performance weekly using `tracking/weekly-kpi-dashboard.csv`.
